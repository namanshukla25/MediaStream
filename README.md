# MediaStream

A web-based video streaming platform developed using PHP and MySQL.

## Technologies
- PHP
- MySQL
- HTML/CSS
- XAMPP

## Features
- Upload videos
- View videos
- Play videos
- Delete videos

## Database Setup
1. Create database in phpMyAdmin
2. Import database.sql
3. Update db.php credentials

## Run Project
Place project inside:
xampp/htdocs

Open:
localhost/MediaStream
